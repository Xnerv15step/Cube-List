基本結構

標準魔術方塊（Rubik's Cube）是一個 3×3×3 的立體拼圖，由奧地利建築師 Ernő Rubik 於 1974 年發明。整個方塊由 26 個可見的小方塊組成，加上中心軸機構共 27 個單元。

方塊分為三種類型的零件：中心塊（Center）、稜塊（Edge）和角塊（Corner）。中心塊永遠固定代表該面的顏色；稜塊有兩個顏色面；角塊有三個顏色面。

六個面各有一個顏色，標準配色為：白對黃、橘對紅、綠對藍。了解這個結構是還原方塊的基礎。

  • 中心塊（Center Pieces）：6 個，固定位置，決定各面顏色
  • 稜塊（Edge Pieces）：12 個，位於各稜邊，有 2 個顏色
  • 角塊（Corner Pieces）：8 個，位於各角落，有 3 個顏色
// 方塊狀態的數學統計 總排列數 = 43,252,003,274,489,856,000 ≈ 4.3 × 10^19（約 430 億億種狀態） God's Number（神之數）= 20 // 任何狀態最多 20 步（HTM）即可還原

魔術方塊的群論（Group Theory）讓我們能精確描述每個狀態與操作的數學關係，是現代速解演算法的理論基礎。

符號系統

魔術方塊的標準符號系統（Notation）由 David Singmaster 於 1979 年提出，至今仍是全球通用的溝通語言。每個字母代表一個面的旋轉操作。

基本六個面的縮寫分別為：U（上）、D（下）、L（左）、R（右）、F（前）、B（後）。

符號的修飾規則如下：

  • 單獨字母（如 R）：順時針旋轉 90°
  • 加撇號（如 R'）：逆時針旋轉 90°
  • 加數字 2（如 R2）：旋轉 180°
  • 小寫字母（如 r）：雙層旋轉（Wide Move）
  • M、E、S：中間層旋轉
// 常用符號對照表 U → 上層順時針 U' → 上層逆時針 U2 → 上層 180° D → 下層順時針 D' → 下層逆時針 D2 → 下層 180° R → 右層順時針 R' → 右層逆時針 R2 → 右層 180° L → 左層順時針 L' → 左層逆時針 L2 → 左層 180° F → 前層順時針 F' → 前層逆時針 F2 → 前層 180° B → 後層順時針 B' → 後層逆時針 B2 → 後層 180° M → 中層（同 L） E → 中層（同 D） S → 中層（同 F）

熟悉符號系統後，就能讀懂任何速解教學與競賽還原解法，這是學習魔術方塊的第一步。

還原方法

還原魔術方塊有多種方法，適合不同程度的玩家。最常見的入門方法是「層先法」（Layer by Layer），顧名思義就是一層一層地還原方塊。

CFOP 是目前世界主流的速解方法，由 Jessica Fridrich 發展完善，縮寫代表四個步驟：Cross（十字）、F2L（下兩層）、OLL（頂層朝向）、PLL（頂層排列）。

Roux 方法是另一種流行的速解系統，著重於減少移動步數並大量使用 M 層旋轉，廣受高階選手喜愛。

💡 建議新手先學層先法，平均約 100 步還原；熟練後再學習 CFOP，平均只需 55～60 步。

常見還原方法比較：

  • 層先法（Beginner's Method）：步驟多但直觀，適合初學者
  • CFOP：主流速解法，WCA 世界紀錄保持者多採用此法
  • Roux：效率高、手法獨特，學習曲線較陡
  • ZZ：預規劃 EO，適合電一手法的選手

演算法公式

公式（Algorithm）是一串特定的轉動序列，用來在不破壞已完成部分的情況下改變特定方塊位置。每個公式針對特定情況設計，熟記常用公式是提升速度的關鍵。

以下是幾個必學的基礎公式，涵蓋從初學到進階的各個階段：

F2L 右手配對（Right-Hand Slot）：

R U R'

F2L 左手配對（Left-Hand Slot）：

L' U' L

OLL 頂層十字（Dot Case）：

F R U R' U' F' f R U R' U' f'

PLL T 排列（T-Perm）—— 交換對角稜塊與角塊：

R U R' U' R' F R2 U' R' U' R U R' F'

公式練習建議：先理解原理再記憶，每天反覆練習直到形成肌肉記憶。熟練的速解選手能在 0.3 秒內執行一個完整公式。

進階技巧

達到 sub-1 分鐘（一分鐘內還原）後，進一步提升速度需要掌握進階技巧，這些技巧涵蓋手法效率、視覺追蹤與策略規劃。

Look-ahead（前瞻）是最重要的進階技能：在執行當前公式時，眼睛同時追蹤下一個需要處理的方塊，避免「停頓思考」的時間浪費。

Fingertricks（手指技巧）直接影響 TPS（每秒轉動次數）。高效的手指技巧包括：

  1. U 層旋轉盡量使用食指或中指，而非整個手掌
  2. 右手 R 移動使用中指推動，比食指更快
  3. 練習雙手同步操作，左右手均衡發力
  4. D 層移動盡可能用 Wide Move 代替以減少換手
  5. 連續相同方向的旋轉應一氣呵成不換手
🏆 目前 3×3 魔術方塊世界紀錄：Max Park 於 2023 年創下 3.13 秒的單次還原紀錄。

X-Cross（擴展十字）是 F2L 的高級變體，在製作白色十字的同時一併完成第一個 F2L 配對，等同於節省了整個 F2L 配對的時間。
