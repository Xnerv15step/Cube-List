# 🧩 魔術方塊技術文件頁面


<img width="1920" height="914" alt="魔術方塊技術文件 - Google Chrome 2026_2_20 上午 01_51_25" src="https://github.com/user-attachments/assets/7c2ef0e5-eacd-422f-a128-447317fc5928" />




一個以深色科技風格設計的魔術方塊（Rubik's Cube）技術文件靜態網頁，涵蓋從基礎結構到進階速解技巧的完整說明。

---

## 📁 專案結構

```
rubiks-doc/
├── index.html    # 頁面結構與內容
├── style.css     # 所有樣式規則
└── README.md     # 專案說明文件
```

---

## 🌟 功能特色

- 固定左側導覽列，點擊可跳至對應章節
- 響應式設計，支援桌面與行動裝置
- 深色科技風 UI，搭配藍色主題配色
- 程式碼區塊高亮顯示
- 章節標籤（Badge）與提示框（Tip Box）視覺強調

---

## 📖 文件章節

| 章節 | 內容摘要 |
|------|----------|
| 基本結構 | 方塊零件組成、排列數統計、群論簡介 |
| 符號系統 | Singmaster 符號全覽、修飾規則說明 |
| 還原方法 | 層先法、CFOP、Roux、ZZ 方法比較 |
| 演算法公式 | F2L、OLL、PLL 必備公式列表 |
| 進階技巧 | Look-ahead、手指技巧、訓練計畫 |

---

## 🚀 使用方式

無需安裝任何套件，直接以瀏覽器開啟即可。

```bash
# 複製專案
git clone https://github.com/your-username/rubiks-doc.git

# 進入資料夾
cd rubiks-doc

# 以瀏覽器開啟
open index.html
```

> 或直接雙擊 `index.html` 以預設瀏覽器開啟。
